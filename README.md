# Host local to test changes before deployment
(If WindowS)
- Install Choco
- Install Hugp
- Install Git
- Install Go

After installing all of these, navigate to your Hugo server folder, and run:

```bash
hugo server
```

Look at the locally hosted server at http://localhost:1313
